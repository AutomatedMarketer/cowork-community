# Post block template

Use this format when showing the one daily strategic post for approval.

```markdown
### Today's strategic post — {{DAY_OF_WEEK}}: {{ROTATION_TYPE}}

**Where this posts:** {{COMMUNITY_NAME}}, {{SPACE_OR_CHANNEL_IF_APPLICABLE}}
**Post type:** {{ROTATION_TYPE — e.g. "Repurpose YouTube clip" or "Engagement question"}}
**Pulled from / inspired by:** {{SOURCE — link to YouTube video, SOP, prior post, etc.}}

**Proposed post:**

{{POST_BODY}}

**Length:** {{N}} sentences, {{N}} paragraphs.
**Funnel touch included:** {{YES_WITH_DETAIL | NO_BECAUSE_POST_TYPE_DOES_NOT_NEED_ONE}}
**Has CTA / question:** {{YES + WHAT_IT_ASKS | NO}}
**Voice rating (my gut check):** {{ON | OFF | NOT_SURE}}

---

⏸ Approve this post? (yes / edit / skip)
```

## Rendering rules

- Only one strategic post per daily run. If the rotation says "off" today, the agent skips this block entirely.
- Show the source — repurposed posts should make the source visible to the user.
- "Funnel touch" handling differs from replies. Posts often ARE the touch.
- CTA / question flag helps the user see if the post will drive engagement.
- Voice rating self-assessment — same as reply blocks.
